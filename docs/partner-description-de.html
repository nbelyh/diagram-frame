Diagrammrahmen SharePoint Visio-Webpart

Das Webpart ist ein spezielles Widget zum Anzeigen eines Microsoft Visio-Diagramms auf einer modernen SharePoint-Seite. Es ähnelt in gewisser Weise dem Webpart „Visio Web Access“ im „klassischen“ SharePoint.

Zusätzlich zum Standard-Webpart "File Viewer", mit dem Sie ein Visio-Diagramm anzeigen können, bietet dieses Folgendes:

Dateibrowser

Dateibrowser, spezifisch für Visio-Diagramme

Startseite

Sie können festlegen, welche Seite zuerst angezeigt werden soll. Sie können beide Seitennamen angeben (z. B. "Meine zweite Seite"). Dies funktioniert nur, wenn das Diagramm mehr als eine Seite hat. Falls die ausgewählte Seite nicht existiert, wird standardmäßig die erste Seite angezeigt.

Standard-Zoomstufe auswählen

Sie können die Standard-Zoomstufe für das Diagramm festlegen.

Unterstützung für volle Breite und Anwendungsseitenmodus

Sie können das Webpart als Anwendungsseite hinzufügen.

Unterstützung für benutzerdefinierte Breiten-/Höheneinstellungen

Sie können die Breite und Höhe des Diagramms festlegen. Kann in Prozent (auch Prozent des Bildschirms) oder tatsächlichen Maßeinheiten (wie die klassische „Visio-Diagrammansicht“) angegeben werden.

Unterstützt das Deaktivieren von Menüelementen

Sie können Diagrammsymbolleiste, Diagrammgrenze und Rahmen optional ausblenden

Unterstützen Sie das Deaktivieren einiger Interaktivität

Optional können Sie Hyperlinks, Zoom, Pan und den Navigator deaktivieren

Was ist neu in Version 1.2

  • Deutsche Lokalisierung
  • Optionale Navigation zwischen Seiten und Diagrammen mit einem Klick (experimentell)

Was ist neu in Version 1.1

– Ermöglicht die Verwendung des Webparts als Seite in voller Breite – Ermöglicht das Einbetten der Registerkarte als Team-Registerkarte

  • Optimiertes Flackern beim ersten Laden
  • Optimiertes Erlebnis beim Hochladen von Dateien (Standardwerte für das Hochladen in die Dokumentbibliothek der Website)
